# VERSION 1 — Clean & Minimal

**Feel:** Premium, minimal, easy to read from 3–5 feet. Lots of air.

## 1. Document setup (Pages)

- **File → Page Setup:** Paper Size **US Letter**, Orientation **Portrait**.
- **Document → Document** (or Format → Document):  
  - Margins: **0.75"** top, bottom, left, right (or 1" if you prefer more margin).
- **View → Show Layout** so you can see margins.

## 2. Background color

- **Format → Document → Document** (or use Format inspector).
- Under **Background**, choose **Color**.
- Set a custom color: **Off-white / light cream**.  
  - In Color Picker: try **RGB 252, 250, 245** or **255, 251, 243** (very light cream).

## 3. Optional thin page border

- **Insert → Shape → Rectangle**.
- Drag to fill the page inside margins.
- **Format → Shape → Style:**  
  - Fill: **None**.  
  - Border: **Line**, 0.5–1 pt, color **muted tan** (e.g. RGB 200, 190, 178).  
- **Arrange → Send to Back** so it sits behind text.

## 4. Typography and spacing

Create **one text box** (or use the default page and type there). Paste the full menu text into it, then:

**Header block**

- **UNSERIOUS COFFEE**  
  - Font: **Helvetica Neue Bold** or **SF Pro Display Bold** (or **Arial Bold**).  
  - Size: **28–32 pt**.  
  - Color: **Charcoal** — e.g. RGB 45, 45, 45 or 55, 52, 48 (deep brown-black).  
  - Alignment: **Center**.  
  - Space **after** paragraph: **6–8 pt**.

- **Life's Not That Serious.**  
  - Font: **Helvetica Neue Light** or **Italic** (or **Georgia Italic** for a serif option).  
  - Size: **14–16 pt**.  
  - Color: same charcoal or slightly lighter (e.g. RGB 80, 75, 70).  
  - Alignment: **Center**.  
  - Space **after** paragraph: **24–32 pt** (generous gap before divider).

**First divider**

- Insert a **thin horizontal line**: **Insert → Shape → Line**, or type a row of em dashes and style as 0.5–1 pt rule.  
- Color: **Muted tan / warm beige** — e.g. RGB 200, 190, 178 or 185, 175, 165.  
- Width: about **4–5"** centered, or span between margins.  
- Space **below** divider: **28–36 pt**.

**Drink list** (each line is one paragraph)

- Font: **Helvetica Neue** or **SF Pro Text** or **Arial** (Regular).  
- Size: **18–20 pt**.  
  - If you want maximum readability from 3–5 ft, use **20 pt**.  
- Color: **Charcoal** — same as header (RGB 45, 45, 45 or 55, 52, 48).  
- Alignment: **Center**.  
- **Line spacing:** 1.2–1.4 (or **Space after** each line: **14–18 pt** for even vertical rhythm).  
- No bullet points; one drink per line.

**Second divider**

- Same as first: thin line, muted tan, centered.  
- Space **above**: **20–24 pt**. Space **below**: **20–24 pt**.

**Footer line**

- **All drinks available Hot or Iced.**  
  - Font: **Helvetica Neue Light** or **Regular**, **12–14 pt**.  
  - Color: **Slightly lighter** than body — e.g. RGB 100, 95, 90.  
  - Alignment: **Center**.

## 5. Optional skeleton watermark (Version 1)

- Find a **minimal skeleton icon** (e.g. small skull or single bone) — PNG with transparent background.
- **Insert → Choose** (image). Place behind header.
- **Format → Image → Opacity:** **4–8%**.  
- Position: centered, just above or behind “UNSERIOUS COFFEE.”  
- **Arrange → Send to Back** so all text sits on top.

# VERSION 2 — Slightly Bolder (More Personality)

**Feel:** Same premium and minimal base, but a bit more character and presence.

## Differences from Version 1

**Header**

- **UNSERIOUS COFFEE**  
  - Font: **Helvetica Neue Heavy** or **Black**, or **Georgia Bold**.  
  - Size: **32–36 pt** (slightly larger).  
  - Same charcoal color.  
  - Optional: **Letter spacing** widened by 1–2 pt for a more “display” look.

- **Life's Not That Serious.**  
  - Slightly larger: **16–18 pt**.  
  - Use **Georgia Italic** if you want a serif contrast, or keep sans italic.  
  - Same spacing idea: big gap after tagline.

**Dividers**

- Use a **slightly thicker** rule (1–1.5 pt) in the same muted tan.  
- Or use a **double thin line** (two close lines) for a bit more graphic punch.

**Drinks**

- Font: **Helvetica Neue Medium** or **SemiBold** (not Regular) so names stand out a bit more.  
  - Or keep Regular but bump size to **20–22 pt**.  
- Same center alignment and generous line spacing.

**Footer**

- **All drinks available Hot or Iced.**  
  - Same as Version 1, or **1 pt larger** and same muted color.

**Optional extras (keep minimal)**

- **Corner accents:** Very small (e.g. 0.5 pt) L-shaped lines in the corners, same tan, inside the margin.  
- **Skeleton watermark:** Same as Version 1 but at **6–10%** opacity so it’s a bit more visible.

# Checklist before printing

- [ ] Page size 8.5" × 11", Portrait  
- [ ] Margins 0.75"–1"  
- [ ] Background off-white/cream  
- [ ] No prices, no sizes  
- [ ] All drinks centered, one size only  
- [ ] “All drinks available Hot or Iced.” at bottom  
- [ ] Print at 100% scale (no “Fit to page”)  
- [ ] Use “Print Background” (or equivalent) so the cream color prints  

You’re done. Simple, premium, minimal — and ready for Apple Pages and the printer.
